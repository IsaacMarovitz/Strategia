# Strategia
 
The backup for the new turn-based strategy game I'm working on. It's called Strategia and is based off of Civilisation VI and the 90s game Strategic Conquest. This project uses Unity 2021.1.16f1. The Trello page is here: https://trello.com/b/g2jaVjZD/strategia.
